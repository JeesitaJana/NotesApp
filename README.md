<h2>NotesApp</h2>
<p>
This is a beginner-level Django project created to understand the core concepts of Django by building a simple Notes application. <br>
The app allows users to: <br>

<ul>
<li> View a welcome homepage</li>

<li>Add notes with a title and content</li>

<li>View all saved notes</li>

<li>Manage notes through the Django Admin panel</li>
</ul>

This project focuses on learning Django fundamentals rather than advanced features. <br> <br>
<h3>Tech Stack </h3>
<ul>
<li>Python</li>

<li>Django</li>

<li>SQLite (default Django database)</li>

<li>HTML (Django Templates)</li>
</ul>
